# fake-news-detection-app
Fake News Detection System using TF-IDF and MLP (AIML Project)

## Overview
A machine learning system that classifies news as **Real or Fake** using NLP techniques.

## Technologies Used
- Python
- Scikit-learn
- TensorFlow / Keras
- Streamlit

## Model
- TF-IDF Vectorization
- Multilayer Perceptron (MLP)
- Manual Hyperparameter Tuning

## Results
- Accuracy: ~98.9%
- F1 Score: ~0.99
- ROC-AUC: ~0.999

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
